Eventos Condicionales y Bucles en JavaScript

Esta página web permite al usuario ingresar un número y comprobar si es mayor o menor a 10. Además, la aplicación mostrará una lista de números desde el 1 hasta el número ingresado. Esta funcionalidad utiliza condicionales para evaluar el número y bucles para generar la lista de números. También maneja entradas inválidas para asegurar que el usuario ingrese un valor correcto.
